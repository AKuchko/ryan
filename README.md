# ryan

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run server
```
npm run start
```

## Project Stack
### Frontend
- Vue.js 3.2 

### Backend
- Express (Web-Framework for Node.js apps)
- MongoDB (Database)
- Mongoose (ORM)
- Morgan (HTTP logger)
